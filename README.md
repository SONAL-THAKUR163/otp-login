#OTP Login Authentication System

A simple OTP (One-Time Password) Login Authentication System built using HTML, CSS, JavaScript, Node.js, and Express.js.

## 🚀 Features

- User-friendly login page
- Email OTP generation
- OTP verification
- Simple and responsive UI
- Backend built with Node.js and Express
- Easy to understand project structure

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Node.js
- Express.js
- Nodemailer

## 📁 Project Structure

```
otp-login/
│── node_modules/
│── home.html
│── index.html
│── script.js
│── style.css
│── server.js
│── package.json
│── package-lock.json
```

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/SONAL-THAKUR163/otp-login.git
```

2. Open the project folder

```bash
cd otp-login
```

3. Install dependencies

```bash
npm install
```

4. Start the server

```bash
node server.js
```

5. Open your browser and visit

```
http://localhost:3000
```
